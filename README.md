# Pokemon 3D Word Quest

Standalone browser game for spelling and listening practice.

## Student Link

https://2021vivasupereducation-boop.github.io/pokemon-word-quest/

## Teacher Mode

Add `?teacher=1` to the game URL to show Google Sheet settings and sync tools.
