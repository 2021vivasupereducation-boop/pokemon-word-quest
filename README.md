# Pokemon 3D Word Quest

Standalone browser game for spelling and listening practice.

## Student Link

After GitHub Pages is enabled, send parents the GitHub Pages URL for this repository.

## Teacher Mode

Add `?teacher=1` to the game URL to show Google Sheet settings and sync tools.

## Google Sheet

Scores are sent to Apps Script and appended to a separate worksheet tab:

`Pokemon 3D Word Quest`

The Apps Script code is in `google-sheet-leaderboard-apps-script.gs`.
